# generative-article
Generate Article using Gemini AI and Pexels.
TechStack used :
  * Node JS
  * Express JS
  * Google Gemini AI API
  * Pexels API

run app using
```sh
node app.js
```
or using nodemon
```sh
npm run start
```

access endpoint using `POST` method at 
```sh
http://localhost:3000/
```
body 
```
{
 keyword : "ARTICLE_KEYWORD"
}
```
